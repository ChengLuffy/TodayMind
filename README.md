# 🤔 What is TodayMind
`TodayMind` is a tiny project that provides amazing ability to manage `Reminders` directly in Today Widget

# 🤘 Features
- Text inputting on widget
- Full-featured widget with ultimate swipe gestures
- EventKit usages
