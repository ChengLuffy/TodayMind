# 🤔 TodayMind 是什么
`TodayMind` 是一个非常小的项目，提供了直接在通知中心管理 `提醒事项` 的神奇能力

# 🤘 特性
- 直接在通知中心完成输入文字 -> 创建提醒事项的过程
- 没有任何限制的通知中心小部件体验，包括滑动等手势操作
- EventKit API 的使用
